¿Como se define una función?

Como una correlación entre el dominio de la funcion y el codominio de la misma.

Como una relación binaria, de forma que a cada elemento x del dominio le corresponde un único elemento del codominio

Como una relación binaria, de forma que a cada elemento x del codominio le coresponte un unico elemento del dominio.

Como una relacion, de forma que a cada elemento x del dominio le corresponde uno o varios elementos del codominio.
